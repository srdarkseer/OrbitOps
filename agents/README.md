# OrbitOps Agents

This directory contains the implementation of all 7 specialized agents:

- `agent1_architecture_reader.py` - Reads cloud architecture
- `agent2_efficiency_analyzer.py` - Finds inefficiencies
- `agent3_cost_simulator.py` - Runs cost simulations
- `agent4_terraform_generator.py` - Generates Terraform
- `agent5_safe_deployer.py` - Applies changes safely
- `agent6_documentation_writer.py` - Writes documentation
- `agent7_monitor.py` - Monitors logs & anomalies
